ผมออกแบบพอร์ตโฟลิโอนี้เพื่อสื่อสารความสามารถด้านการพัฒนาเว็บอย่างชัดเจน โดยให้ความสำคัญกับการนำเสนอข้อมูลที่อ่านง่าย เป็นระเบียบ และสะท้อนตัวตนในฐานะผู้สมัครฝึกงานสายเทคโนโลยี

ปัจจุบันผมเป็นนักศึกษาที่สนใจงานพัฒนาเว็บแบบ Fullstack โดยมีประสบการณ์ในการเชื่อมต่อระบบหน้าบ้านและหลังบ้านด้วย Laravel, Blade, JavaScript และ MySQL รวมถึงสามารถออกแบบ Business Rules, Validation และสถานะเอกสารให้สอดคล้องกับการใช้งานจริง

Fullstack Workflow

สิ่งที่ผมทำได้จริงในงาน Fullstack

Frontend

พัฒนา UI ด้วย Blade + JavaScript ให้ฟอร์มใช้งานง่ายและลดการกรอกผิด

Backend

เขียน Laravel Controller และ Service เพื่อบังคับ business rules ตามเงื่อนไข

Database

ออกแบบตาราง MySQL ให้รองรับความสัมพันธ์ข้อมูลและติดตามประวัติได้

Status Flow

กำหนดสถานะเอกสารให้ชัดเจนตั้งแต่ร่างจนถึงอนุมัติหรือยกเลิก

input >> validation >> process >> save >> update status
Business Rules

Validation

ตรวจ required, format และเงื่อนไขซ้อนก่อนบันทึก เพื่อให้ข้อมูลถูกต้อง
